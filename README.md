# Сокращатель ссылок (API clc.to)

Скрипт позволяет сокращать длинные URL или проверять количество кликов по уже созданной короткой ссылке через сервис clc.li (clc.to).

### Функционал
* Если ввести обычную ссылку — скрипт вернет её короткую версию.
* Если ввести уже сокращенную ссылку — скрипт покажет количество переходов по ней.

### 🛠 Как установить
1. **Скачайте код:**
```bash
git clone https://github.com/skislyakow/clc.ci
cd clc.ci
```

### Установите зависимости:
Для работы нужен Python 3 и библиотека `requests` и : `python-dotenv`
```bash
pip install requests python-dotenv
```
или запустите
```bash
pip install -r requirements
```

### Настройте переменные окружения:
Создайте в корне проекта файл `.env` и добавьте туда свой API-токен от сервиса `clc.to`:
```env
CLCLI_TOKEN=ваш_токен_здесь
```

### 🚀 Запуск
Запустите скрипт через терминал:
```bash
python clc.li.py [url]
```
Если url передан аргументом — скрипт сразу обработает ссылку.
Если аргумент не передан — предложит ввести ссылку вручную.

**Пример работы:**
```bash
python clc.li.py https://example.com/long/url
python clc.li.py https://clc.li/abc123
python clc.li.py  # режим ввода вручную. После запуска введите нужную ссылку.
```
